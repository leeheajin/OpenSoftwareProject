Open-Software Project (01)
Assignment9 : OSP14-CNN Architecture

 Python3과 Pycham, Anaconda prompt 환경에서 이를 구현하였다.

1. main.py
  실제 함수를 시작하는 부분이다. torchvision.datasets.CIFAR10 데이터셋을 다운받아 이용할 수 있다.
  불러오는 model과 PATH 부분을 조정해 어떠한 모델을 이용할 것인지를 결정한다.

2. vgg16_full.py
  실제 구현할 부분은 없으며 해당 코드를 이용해 쉽게 main에서 VGG 모델을 사용할 수 있다.

3. resnet50_skeleton.py
  실제 구현할 코드가 빈칸으로 표기되어 있다. ResNet을 구현하기 위한 변수들과 함수들의 인자를 넣어서 구현한다.
  메인 함수에서 주석 처리된 부분을 다시 바꾸고, VGG에서 이용했던 부분을 주석처리 해 실행할 수 있다.
